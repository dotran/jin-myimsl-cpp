plhls: Set current color by HLS

plhls (h, l, s);

Set current color by hue, lightness, and saturation. Convert hls color coordinates to rgb, then call plrgb. Do not use this. Only retained for backward compatibility. Use plhlsrgb and plscol0 instead.
