plgfci: Get FCI (font characterization integer)

plgfci (fci);

Gets information about the current font using the FCI approach. See the Section called FCI in Chapter 3 for more information.

fci (PLUNICODE *, output)

Pointer to PLUNICODE (unsigned 32-bit integer) variable which is updated with current FCI value.
