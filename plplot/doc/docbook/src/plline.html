plline: Draw a line

plline (n, x, y);

Draws line defined by n points in x and y.

n (PLINT, input)

Number of points defining line.

x (PLFLT *, input)

Pointer to array with x coordinates of points.

y (PLFLT *, input)

Pointer to array with y coordinates of points.
