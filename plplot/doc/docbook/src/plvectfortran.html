plvect: Vector plot, fixed linear mapping for fortran

plvect (u, v, nx, ny, scale);

When called from Fortran, this routine has the same effect as when invoked from C. The interpretation of all parameters (see plvect) is also the same except there is no transformation function supplied as the last parameter. Instead, a 6-element array specifying coefficients to use in the transformation is supplied via the named common block plplot (see code). Since this approach is somewhat inflexible, the user is recommended to call either of plvec0, plvec1, or plvec2 instead.
