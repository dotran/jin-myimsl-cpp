plglevel: Get the (current) run level

plglevel (p_level);

Get the (current) run level. Valid settings are:

  • 0, uninitialized

  • 1, initialized

  • 2, viewport defined

  • 3, world coords defined

p_level (PLINT *, output)

Pointer to the run level.
